Checking local server...

The app can run from a local server so browser features behave more like a real app. Calendar drafts and .ics files work now; automatic Google/Outlook syncing is the next backend step.

  1. Google: create a Google Cloud project, enable Calendar API, configure OAuth, and add this local URL as an authorized origin.
  2. Outlook: register a Microsoft Entra app, add Graph calendar permissions, and use OAuth authorization code flow.
  3. Production: store provider tokens securely and save provider event IDs on each job.
